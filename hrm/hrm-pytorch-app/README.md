## Installation

To set up the project, clone the repository and install the required dependencies:

```bash
git clone https://github.com/fionntmcc/Final-Year-Project
cd hrm-pytorch-app
pip install -r requirements.txt
```

## Usage

To run the application, execute the following command:

```bash
python src/main.py
```

This will initialize the application, load the data, train the model, and evaluate its performance.
